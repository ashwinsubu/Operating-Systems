#Dining Philosophers Problem

The classic dining philosophers problem faces deadlock situation and 
this program demonstrates the problem with deadlock and deadlock avoidance.
Each fork is a semaphore and we fix the deadlock by making one philosopher pick the 
right fork while others pick the left one.


_____Steps to Run________
Compile:
javac DiningPhilosopherProblem.java

Run: 
java DiningPhilosopherProblem
